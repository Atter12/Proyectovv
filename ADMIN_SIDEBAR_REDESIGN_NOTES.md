# Admin Sidebar Redesign Notes

Se aplicó un rediseño real del sidebar administrativo con una paleta más sobria y enterprise.

## Cambios principales

- Paleta ajustada a azul petróleo oscuro + jade apagado.
- Verde limitado a estados activos, éxito y acción principal.
- Reducción del efecto glow para evitar sensación gaming/template.
- Nuevos tokens CSS para sidebar: fondos, surfaces, textos, bordes y estados.
- Sidebar tratado como componente dedicado con panel interno, brand card, estado operativo, navegación agrupada y acción rápida.
- Estados UX refinados: hover, active, focus-visible y pressed.
- Mejor contraste tipográfico para labels, descripciones, indicadores y secciones.
- Bordes y divisores más sutiles para una apariencia premium.
