# Admin UI v2 Recommendations

This package includes the original project plus a design brief for the next UI iteration.

Recommended palette:
- Background: #081C2A
- Sidebar: #0E2A3A
- Surface: #14384D
- Hover: #1C455E
- Accent: #74D3B4
- Success: #59C493
- Warning: #F4C95D
- Danger: #E76F6F

Sidebar improvements:
- Remove heavy glow.
- Reduce accent usage to active item and success states.
- Softer borders (rgba(255,255,255,.08)).
- Better typography hierarchy.
- Group navigation by domain.
- Add subtle motion (150–220ms).
